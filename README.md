# Project-3-Sentiment-Analysis-of-Reviews
Python project that collects 60+ movie reviews from The Movie Database (TMDb) and performs sentiment analysis, word frequency analysis, and a word cloud visualization.
##### Tools Used:
Python,
Jupyter Notebook,
TMDb API,
TextBlob,
Matplotlib,
WordCloud

##### How to Run:
1. Add your TMDb API key in the notebook where indicated.
2. Install the required Python libraries.
3. Run the notebook top to bottom.

##### Summary
The project gathers at least 60 reviews from popular movies and analyzes whether the reviews are positive, negative, or neutral. It also removes stopwords and shows the top words in a WordCloud.
